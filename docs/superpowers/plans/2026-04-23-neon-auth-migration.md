# Neon Auth Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Better Auth with Neon Auth so users are managed in the Neon console, identity lives in `neon_auth.users_sync`, and Better Auth is fully removed.

**Architecture:** Neon Auth owns identity (sign-up / sign-in / session tokens). The app keeps an app-owned `user_profiles` table (text PK) that logically references `neon_auth.users_sync.id` and stores `household_id`. Server functions resolve the session via Neon's server SDK, then lazy-provision a profile + household on first authenticated request.

**Tech Stack:** TanStack Start + React 19 + Vite 8 SSR, Drizzle ORM 0.45.2, Neon serverless Postgres, pnpm, Playwright (e2e), Neon Auth React + server SDKs.

**Source spec:** [2026-04-23-neon-auth-migration-design.md](../specs/2026-04-23-neon-auth-migration-design.md)

**Dev-only migration:** no production user data. `db:push` drop/create freely; run `pnpm seed --reset` after the schema lands.

import { eq, sql } from "drizzle-orm"
import { db } from "@/server/db"
import { households, userProfiles } from "@/server/db/schema"
import { getSessionUser } from "@/server/auth"

export type Profile = {
  userId: string
  householdId: number
}

export async function requireProfile(userId: string): Promise<Profile> {
  const existing = await db.query.userProfiles.findFirst({
    where: eq(userProfiles.userId, userId),
    columns: { userId: true, householdId: true },
  })
  if (existing) return existing

  return db.transaction(async (tx) => {
    const [household] = await tx
      .insert(households)
      .values({ name: "My Household" })
      .returning({ id: households.id })

    const inserted = await tx
      .insert(userProfiles)
      .values({ userId, householdId: household.id })
      .onConflictDoNothing({ target: userProfiles.userId })
      .returning({ userId: userProfiles.userId, householdId: userProfiles.householdId })

    if (inserted.length > 0) return inserted[0]

    // Concurrent request won the race. Roll back our orphaned household and read theirs.
    await tx.delete(households).where(eq(households.id, household.id))
    const winner = await tx.query.userProfiles.findFirst({
      where: eq(userProfiles.userId, userId),
      columns: { userId: true, householdId: true },
    })
    if (!winner) throw new Error("Profile race resolution failed")
    return winner
  })
}

export async function withProfile<T>(
  fn: (ctx: Profile) => Promise<T>,
): Promise<T> {
  const session = await getSessionUser()
  if (!session) throw new Error("Unauthorized")
  const profile = await requireProfile(session.userId)
  return fn(profile)
}

// Keep the `sql` import alive for the tx type inference (drizzle quirk).
